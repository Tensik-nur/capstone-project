# CyberShield Academy - Образовательная платформа по кибербезопасности

## 📋 Описание проекта

CyberShield Academy - это современная образовательная платформа нового поколения, специализирующаяся на обучении кибербезопасности. Проект разработан с фокусом на безопасность и предоставляет структурированные курсы, практические лаборатории и интерактивные квесты.

## 🏗️ Структура проекта

```
project capstone 1/
├── index.html                          # Главная страница
├── css/
│   └── main.css                       # Основные стили для всех страниц
├── js/
│   └── main.js                        # Основной JavaScript для функциональности
├── assets/
│   ├── images/                        # Папка для изображений
│   └── icons/                         # Папка для иконок
└── pages/
    ├── modules.html                   # Главная страница модулей
    ├── education/                     # Демонстрационные лекции и материалы
    │   ├── basics.html               # Основы кибербезопасности
    │   ├── cryptography.html         # Криптография
    │   ├── network-security.html     # Сетевая безопасность
    │   └── defense.html              # Защита от угроз
    ├── practical/                     # Практические упражнения
    │   ├── labs.html                 # Практические лаборатории
    │   ├── pentesting.html           # Этичный хакинг и тестирование
    │   └── game-cyber.html           # Интерактивная кибер-игра
    ├── community/                     # Раздел сообщества
    │   └── community.html            # Страница сообщества
    ├── legal/                         # Законодательный раздел
    │   └── law.html                  # Законодательство и стандарты
    └── auth/                          # Аутентификация
        └── register.html              # Страница регистрации
```

## 🎯 Основные разделы

### 1. **Образование (Education)**

- **Основы кибербезопасности** - Фундаментальные знания о защите информации
- **Криптография** - Изучение методов шифрования и алгоритмов
- **Сетевая безопасность** - Защита сетей и протоколов
- **Защита от угроз** - Выявление и предотвращение кибер-атак

### 2. **Практика (Practical)**

- **Лаборатории** - Интерактивные упражнения в изолированной среде
- **Пентестинг** - Тестирование на проникновение и поиск уязвимостей
- **Кибер-квест** - Геймифицированные задания и квесты

### 3. **Сообщество (Community)**

- Форум обсуждения
- Группы интересов
- Вебинары и события
- Возможности трудоустройства

### 4. **Законодательство (Legal)**

- GDPR и локальное законодательство
- Стандарты ISO 27001 и NIST
- Уголовное право
- Этические нормы

## 🎨 Дизайн и визуальный стиль

Сайт использует современный **киберпанк-неон** стиль с:

- Темной готической цветовой схемой
- Неоновыми градиентами (зеленый #39FF14, голубой #00F0FF, фиолетовый #9D4EDD)
- Анимированный пиксельный фон
- Эффекты glitch и pulsе для визуальной привлекательности

## 🔧 Технические особенности

### Frontend

- **HTML5** - Семантическая верстка
- **CSS3** - Современные стили с переменными CSS, анимациями
- **JavaScript ES6+** - Интерактивность и динамическое управление

### Функциональность

- Отзывчивый дизайн (мобильный, планшет, ПК)
- Анимированные элементы и переходы
- Интерактивная игра с пиксельным котом (мотивационный элемент)
- Система терминала с текстом, печатаемым по символам

## 📱 Адаптивность

Сайт полностью адаптирован для:

- **Мобильные устройства** (320px и выше)
- **Планшеты** (768px и выше)
- **Десктопы** (1024px и выше)

## 🔐 Соблюдение безопасности (Концепция)

Хотя это образовательный сайт, структура проекта следует лучшим практикам безопасности:

- Разделение ресурсов (CSS, JS, assets в отдельных папках)
- Клиентская валидация форм
- Структурированный код для легкого добавления серверной валидации
- Защита от XSS через использование textContent вместо innerHTML

## 🚀 Как использовать

### Запуск локально

1. Откройте папку проекта в VS Code
2. Используйте Live Server расширение для просмотра в браузере
3. Или просто откройте `index.html` в браузере

### Навигация

- **Главная** - index.html с основной информацией
- **Модули** - pages/modules.html для выбора курса
- **Практика** - страницы в папке pages/practical/
- **Сообщество** - pages/community/community.html
- **Законодательство** - pages/legal/law.html
- **Регистрация** - pages/auth/register.html
