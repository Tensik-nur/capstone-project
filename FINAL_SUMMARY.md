## 📊 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### На что обратить внимание при редактировании

1. **CSS переменные** - Измените в `:root` для глобального изменения цветов
2. **Пути к файлам** - Используйте относительные пути (../../, ../, ./)
3. **Навигация** - Обновляйте ссылки при добавлении новых страниц
4. **Адаптивность** - Проверяйте на мобильных (320px+)

### Файлы для редактирования

- **Стили** → css/main.css
- **Функциональность** → js/main.js
- **Содержимое** → pages/\*.html
- **Главная страница** → index.html
