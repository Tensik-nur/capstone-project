# 🚀 Быстрый старт - CyberShield Academy

## ⚡ За 5 минут до запуска

### 1️⃣ Открытие проекта

1. Откройте папку проекта в **VS Code**
2. Используйте **Live Server** (правой кнопкой → Open with Live Server)
3. Или откройте `index.html` в браузере

### 2️⃣ Главная страница

Откроется **главная страница** со:

- Логотипом и названием платформы
- Кнопками навигации

- Интерактивным пиксельным котом

## 🆘 Решение проблем

### Стили не загружаются

✅ **Решение:** Проверьте пути в index.html на правильность CSS ссылки

### Ссылки не работают

✅ **Решение:** Убедитесь, что используете Live Server, не просто открыванием файла

### Кот не движется

✅ **Решение:** Нажмите кнопку "Запустить патруль" на главной странице

### JavaScript ошибки в консоли

✅ **Решение:** Откройте DevTools (F12) и проверьте путь к main.js в каждой HTML

## 📝 Редактирование контента

### Изменить текст на странице

1. Откройте нужный .html файл
2. Найдите текст внутри `<h3>`, `<p>` или других тегов
3. Отредактируйте текст
4. Сохраните файл (Ctrl+S)
5. Обновите браузер (F5)

### Добавить новый предмет в образование

1. Скопируйте файл из `pages/education/`
2. Переименуйте его (например, `advanced-topics.html`)
3. Отредактируйте содержимое
4. Добавьте ссылку в `pages/modules.html`

### Изменить цвет дизайна

1. Откройте `css/main.css`
2. Найдите `:root` в начале файла
3. Измените значения цветов в переменных
4. Все цвета на сайте обновятся автоматически

## 🌐 Развертывание на хостинге

### GitHub Pages (БЕСПЛАТНО)

1. Создайте репозиторий на GitHub
2. Загрузите файлы проекта
3. Перейдите в Settings → Pages
4. Выберите main branch
5. Ваш сайт будет доступен по https://username.github.io/

### Netlify (БЕСПЛАТНО)

1. Перейдите на https://netlify.com
2. Кликните "Drop files here" или свяжите GitHub репозиторий
3. Набор настроек не требуется (статический сайт)
4. Получите свой URL

## 📞 Поддержка

### Что-то не работает?

1. Проверьте пути к файлам (особенно css/ и js/)
2. Откройте DevTools (F12) → Console для ошибок
3. Очистите кеш браузера (Ctrl+Shift+Delete)
4. Перезагрузитесь в браузере (Ctrl+Shift+R)
